# CSS Notes вЂ” Styling the Content (MDN Review Notes)

## 1. Purpose of CSS

CSS (Cascading Style Sheets) is used to **style and visually present HTML content**. It defines how elements look rather than what they mean. ([MDN Web Docs][1])

* Controls colors, fonts, spacing, layout
* Works by selecting HTML elements and applying styles
* Separates **structure (HTML)** from **presentation (CSS)**

### Example: HTML with CSS applied

```html
<p>Instructions for life:</p>

<ul>
  <li>Eat</li>
  <li>Sleep</li>
  <li>Repeat</li>
</ul>
```

```css
p {
  font-family: sans-serif;
  color: red;
}

li {
  background-color: greenyellow;
  border: 1px solid black;
  margin-bottom: 5px;
}
```

---

## 2. Applying CSS to HTML

CSS is typically written in a separate file and linked to HTML.

### Step 1: Create CSS file

```css
p {
  color: red;
}
```

Save as:

```
styles/style.css
```

### Step 2: Link CSS in HTML

```html
<link href="styles/style.css" rel="stylesheet" />
```

### Key Points

* `<link>` is placed inside `<head>`
* `href` points to CSS file
* If CSS works correctly в†’ styles reflect in browser ([MDN Web Docs][1])

## 3. CSS Syntax Basics

A CSS rule is called a **ruleset**.

### General Structure

```css
selector {
  property: value;
}
```

### Example

```css
p {
  color: red;
  width: 500px;
  border: 1px solid black;
}
```

### Components

* **Selector** в†’ targets elements (`p`)
* **Declaration** в†’ `property: value`
* **Property** в†’ what to change (`color`)
* **Value** в†’ how to change (`red`)

### Important Rules

* Separate declarations using `;`
* Separate property and value using `:`

### Multiple Selectors

```css
p,
.my-class,
#my-id {
  color: red;
}
```

## 4. Improving Text Styling

### Setting Global Font

```css
html {
  font-size: 10px;
  font-family: "Roboto", sans-serif;
}
```

* Sets base font and size for entire page
* All elements inherit these values ([MDN Web Docs][1])

### Styling Headings and Text

```css
h1 {
  font-size: 60px;
  text-align: center;
}

p,
li {
  font-size: 16px;
  line-height: 2;
  letter-spacing: 1px;
}
```

### Key Properties

* `font-size` в†’ text size
* `text-align` в†’ alignment
* `line-height` в†’ spacing between lines
* `letter-spacing` в†’ space between letters

## 6. CSS Box Model

CSS treats elements as **boxes**.

### Components of a Box

* **Content**
* **Padding** в†’ space inside border
* **Border** в†’ edge around element
* **Margin** в†’ space outside element ([MDN Web Docs][1])

### Example Properties

```css
div {
  width: 500px;
  padding: 20px;
  border: 5px solid black;
  margin: 10px;
}
```

## 8. Styling the Body Element

```css
body {
  width: 600px;
  margin: 0 auto;
  background-color: #ff9500;
  padding: 0 20px 20px 20px;
  border: 5px solid black;
}
```

### Explanation

* `width` в†’ fixed width of content
* `margin: 0 auto` в†’ centers element horizontally
* `padding` в†’ inner spacing
* `border` в†’ outer boundary

### Shorthand Example

```css
padding: 0 20px 20px 20px;
```

Equivalent to:

```css
padding-top: 0;
padding-right: 20px;
padding-bottom: 20px;
padding-left: 20px;
```

---

## 9. Styling the Main Heading

```css
h1 {
  margin: 0;
  padding: 20px 0;
  color: #00539f;
  text-shadow: 3px 3px 1px black;
}
```

### Key Points

* Removes default browser margin
* Adds spacing using padding
* `text-shadow` adds depth effect:

  * Horizontal offset
  * Vertical offset
  * Blur radius
  * Shadow color ([MDN Web Docs][1])

---

## 10. Centering an Image

```css
img {
  display: block;
  margin: 0 auto;
  max-width: 100%;
}
```

### Explanation

* `display: block` в†’ converts inline image to block element
* `margin: 0 auto` в†’ centers horizontally
* `max-width: 100%` в†’ prevents overflow beyond container ([MDN Web Docs][1])

## 11. Key CSS Concepts

### Core Ideas

* CSS applies styles using selectors
* Everything is treated as a **box**
* Layout and spacing controlled via:

  * margin
  * padding
  * border
* Styling includes:

  * colors
  * fonts
  * positioning
